## Sobre mim
 
Eu construo sistemas que transformam dados em decisões.

Engenheiro de Dados e Engenheiro de Software com foco em arquiteturas escaláveis, plataformas de dados e sistemas de alta performance. Projeto e desenvolvo produtos orientados a dados que conectam engenharia, estratégia de negócio e IA para resolver problemas reais.

Minha abordagem vai além do código. Acredito em Clean Architecture como base de sistemas que duram, em dados como ativo estratégico — não apenas armazenamento — e em clareza acima da complexidade. Ownership e disciplina na execução em tudo que construo.

Atuo na interseção entre Engenharia de Dados (Data Lake, ETL, Streaming, Analytics), Backend (APIs, sistemas distribuídos, serverless) e IA & Automação.
